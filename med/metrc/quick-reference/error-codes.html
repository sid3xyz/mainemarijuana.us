System & Connectivity Issues

Handling METRC system-wide problems that affect operations.

Network Problems

  • Slow Response Times: Check local network, try different device, contact ISP.
  • Intermittent Connectivity: Use mobile hotspot as backup, document patterns.
  • Complete Outage: Follow downtime procedures, notify OCP if >4 hours.
  • SSL Certificate Errors: Clear browser cache, check system time, contact METRC support.

Account Issues

  • Locked Account: Wait 15 minutes, reset password, contact admin.
  • Expired Password: Reset immediately, update security questions.
  • Two-Factor Authentication: Check email/SMS, use backup codes, contact support.
  • Role Changes: May take 24 hours to propagate, use alternative access.

Data Entry & Validation Errors

Common mistakes in data entry that cause METRC rejections.

Customer Information

  • Invalid ID Format: Ensure proper state ID format, check expiration.
  • Missing DOB: Required for age verification, enter accurately.
  • Address Errors: Must match ID, use full address format.
  • Duplicate Customers: Search existing records before creating new.

Product Information

  • Weight Discrepancy: Must match package label within tolerance.
  • Strain Mismatch: Verify against batch records and labels.
  • Expiration Issues: Check dates, remove expired products immediately.
  • Package Type Errors: Select correct category (flower, concentrate, edible).

Escalation Procedures

When to escalate errors and who to contact for resolution.

Internal Escalation

  1. Level 1 - Budtender: Try basic troubleshooting, document error.
  2. Level 2 - Shift Lead: Review procedures, attempt advanced fixes.
  3. Level 3 - Manager: Authorize system changes, contact external support.
  4. Level 4 - Owner: Legal/compliance issues, facility-wide problems.

External Contacts

  • METRC Support: 1-855-638-3721 (technical issues, account problems).
  • OCP Compliance: 207-287-4161 (regulatory issues, reporting requirements).
  • OCP IT Helpdesk: help@metrc.com (system outages, data issues).
  • Emergency: 911 (theft, security breaches, immediate threats).

Documentation Requirements

  • Log all errors with timestamps, error codes, and attempted solutions.
  • Record all communications with support and OCP.
  • Retain error logs for 7 years for compliance audits.
  • Include error resolution in staff training sessions.

Error Prevention Best Practices

Proactive measures to reduce METRC errors and improve efficiency.

Staff Training

  • Regular METRC training sessions for all staff.
  • Certification requirements for complex procedures.
  • Cross-training to prevent single points of failure.
  • Performance tracking and coaching for error-prone tasks.

Process Improvements

  • Standardized checklists for common transactions.
  • Double-check procedures for high-value sales.
  • Automated alerts for unusual transactions.
  • Regular equipment maintenance and calibration.

Technology Solutions

  • Use barcode scanners for accurate data entry.
  • Implement POS system integration with METRC.
  • Regular software updates and security patches.
  • Backup internet connections for redundancy.
